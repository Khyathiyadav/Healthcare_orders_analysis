# 📊 Olist E-Commerce Data Analysis & Dashboard

## 🧾 Project Overview
This project analyzes the Olist E-Commerce dataset to uncover key business insights related to sales, customers, delivery performance, and product trends.

The goal is to transform raw data into meaningful insights using Exploratory Data Analysis (EDA) and present them through an interactive Power BI dashboard.

---

## 🎯 Objectives
- Analyze customer purchasing behavior  
- Identify top-selling product categories  
- Understand regional sales distribution  
- Evaluate delivery performance  
- Build an interactive dashboard for business insights  

---

## 📂 Dataset Description
The dataset consists of multiple interconnected tables:

- Orders – Order details and timestamps  
- Customers – Customer location and information  
- Order Items – Products purchased per order  
- Products – Product category details  
- Payments – Payment methods and values  
- Reviews – Customer feedback  
- Sellers – Seller-related information  

---

## 🔍 Exploratory Data Analysis (EDA)

### 🧹 Data Preprocessing
- Handled missing values  
- Removed duplicates  
- Converted data types (timestamps, categories)  
- Merged multiple datasets  

### ⚙️ Feature Engineering
- Extracted year, month, and day from timestamps  
- Created time-based features  
- Derived delivery duration  

### 📊 Analysis Performed
- Sales trend over time  
- Orders distribution by state  
- Popular product categories  
- Payment method usage  
- Delivery time analysis
